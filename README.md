📊 Loan Approval Analysis Dashboard

End-to-End Data Analytics Project

Excel • SQL • Tableau

An end-to-end data analytics project that analyzes loan application data to identify the key factors influencing loan approval decisions. The project uses **Excel** for data preparation, **SQL (PostgreSQL)** for data analysis, and **Tableau** for building an interactive dashboard to support business decision-making.

## 📌 Project Overview

Financial institutions process thousands of loan applications every day. Understanding which applicant characteristics influence loan approvals helps lenders improve lending decisions, identify approval patterns, and monitor portfolio performance.

This project analyzes historical loan application data to answer key business questions using SQL and presents the findings through an interactive Tableau dashboard.

---

## 🎯 Business Problem

A lending institution wants to answer the following business questions:

- What is the overall loan approval rate?
- Which applicant groups have the highest approval rates?
- How does credit history affect loan approval?
- Does education or marital status influence loan approvals?
- Which property areas receive the highest number of approved loans?
- What insights can help improve lending decisions?

## 🎯 Project Objectives

The primary objectives of this project are to:

- Analyze historical loan application data.
- Calculate key business KPIs related to loan approvals.
- Identify factors influencing loan approval decisions.
- Compare approval trends across different applicant segments.
- Build an interactive dashboard for business stakeholders.
- Generate actionable business insights to support data-driven lending decisions.

---

## 📈 Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| Total Loan Applications | Total number of applications received |
| Total Approved Loans | Number of approved applications |
| Total Rejected Loans | Number of rejected applications |
| Loan Approval Rate | Percentage of approved applications |
| Average Loan Amount | Average amount requested by applicants |
| Average Applicant Income | Average income of loan applicants |

## 💡 Key Business Insights

- **Overall Loan Approval Rate:** **68.73%**
- Applicants with a good credit history had significantly higher approval rates.
- Graduate applicants received more loan approvals than non-graduates.
- Married applicants had higher approval counts.
- Semi-urban property areas recorded the highest number of approved loans.
- Credit history was identified as the strongest factor influencing loan approval decisions.

---

## 📌 Business Recommendations

Based on the analysis, the following recommendations can help improve lending decisions:

- Prioritize applicants with strong credit history while maintaining fair lending practices.
- Analyze rejection patterns for applicants with weaker credit profiles.
- Monitor loan approval trends across different property areas.
- Use applicant segmentation to design more targeted lending strategies.
- Track approval KPIs regularly through interactive dashboards.
